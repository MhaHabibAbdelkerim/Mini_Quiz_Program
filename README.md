# Mini_Quiz_App
___ 

A simple Python console-based quiz app that tests your knowledge on Python programming and general questions. The quiz supports multiple difficulty levels and provides immediate feedback after each question.

___

## 🚀 Features

* Multiple difficulty levels: Easy, Medium, and Hard questions.

* Random mode: If no difficulty is selected, questions are shuffled randomly.

* Immediate feedback: Displays “Correct!” or the correct answer if wrong.

* Score tracking: Keeps track of your score and total number of questions.

* Percentage calculation: Shows the final score and percentage at the end.
___

## Example usage
Choose difficulty (easy, medium, hard) to start the quiz: easy
What is the capital of France? Paris
Correct!
